calculus notes / engineering view

用一条主线看懂微积分

函数描述“谁影响谁”，极限描述“逼近到哪里”，导数描述“变化有多快”，微分描述“这一小步变化了多少”，积分描述“把无数小变化合起来是多少”。

导数是率 斜率、速度、边际成本，回答：变化有多快。
微分是量 小位移、小误差、小增量，回答：这一小步变化了多少。
积分是总量 面积、路程、累计收入，回答：很多小量加起来是多少。
极限是桥 从“有限近似”走到“无限精确”的数学工具。
\(f(x)=x^2\) 这一个例子串起全部概念
\(y=f(x)\) 函数：输入 x，输出 y
\(\displaystyle \lim_{x\to a} f(x)=L\) 极限：x 靠近时 y 靠近哪里
\(f'(x)=\frac{dy}{dx}\) 导数：单位 x 变化带来多少 y 变化
\(dy=f'(x)\,dx\) 微分：很小一步 dx 引起的 y 变化
\(\int f(x)\,dx\) 积分：把许多小矩形面积加起来
最重要的区别：导数不是变化量本身，而是变化率；微分才是某一步的近似变化量。速度 \(60\,\mathrm{km/h}\) 是导数，\(0.5\) 小时走了约 \(30\,\mathrm{km}\) 是微分思想。
concept by concept

逐个讲清楚

下面都用尽量工程化的语言解释：它代表什么、问的是什么问题、怎么用一个例子落地。

01

函数：一个量如何决定另一个量

核心刻画：函数建立输入量与输出量之间的确定映射。

函数不只是公式，它更像一台机器：你放入 \(x\)，机器按规则吐出 \(y\)。比如 \(f(x)=x^2\) 表示“输出等于输入的平方”。

  • 它代表什么：变量之间的关系。
  • 它回答什么：给定一个输入，会得到什么输出。
  • 例子：圆面积 \(A(r)=\pi r^2\)，半径 r 决定面积 A。
02

极限：看无限靠近时会怎样

核心刻画：极限描述变量无限逼近时函数值的稳定趋向。

很多时候，直接代入会出问题，或者我们关心的是“越来越接近某点时的趋势”。极限让我们用靠近来定义精确结果。

  • 它代表什么：一个过程的最终趋向。
  • 它回答什么：当输入越来越接近某个值，输出越来越接近什么。
  • 例子：\((x^2-1)/(x-1)\) 在 \(x=1\) 不能直接算，但约分后接近 \(2\)。
03

导数：瞬间的变化有多快

核心刻画：导数是“率”，描述变化有多快，是曲线在某点的斜率。

平均速度是 \(\Delta s/\Delta t\)，瞬时速度就是时间间隔 \(\Delta t\) 趋近 0 时的极限。函数里也一样：导数是“这一点附近，输出对输入有多敏感”。

  • 它代表什么：局部变化率、斜率、速度、敏感度。
  • 它回答什么：变化有多快。
  • 例子：\(f(x)=x^2\)，\(f'(x)=2x\)。在 \(x=3\)，导数是 \(6\)，意思是 x 附近每增加 1，y 大约增加 6。
04

微分：沿着当前斜率走一小步

核心刻画：微分用线性近似刻画自变量微小变化引起的函数变化量。

导数给的是“每单位变化多少”，微分给的是“这一小步实际约变多少”。所以导数像速度表，微分像这一小段路程。

  • 它代表什么：输入小变化 \(dx\) 引起的输出小变化 \(dy\)。
  • 它回答什么：变化了多少。
  • 例子：\(s(t)\) 表示位置，速度 \(v=s'(t)\)。若此刻 \(v=60\,\mathrm{km/h}\)，\(dt=0.5\,\mathrm{h}\)，则 \(ds\approx30\,\mathrm{km}\)。
05

积分：把许多小量加成一个总量

核心刻画：积分描述连续小量在区间上的累积总效应。

积分可以看成很多小矩形的面积相加：每个小矩形是 \(f(x)\,dx\)。当小矩形无限窄，累加结果就变成精确面积。

  • 它代表什么：面积、路程、总收入、总消耗、总概率等累计量。
  • 它回答什么：从一段区间整体看，总共积累了多少。
  • 例子：速度 \(v(t)\) 对时间积分，得到路程 \(s=\int v(t)\,dt\)。
core formulations

每个概念的核心刻画

先把核心语义钉住，再看公式。公式只是把这些朴素问题写得更精确。

function

函数

函数刻画变量之间的映射关系。

输入一个量，按规则得到另一个量：\(x\mapsto f(x)\)。

limit

极限

极限刻画逼近过程中的稳定趋向。

不一定到达，但越靠越近时，看结果趋向哪个值。

derivative

导数

导数刻画函数在一点的瞬时变化率。

斜率、速度、敏感度，回答“变化有多快”。

differential

微分

微分刻画自变量微小变化引起的函数变化量。

用当前切线的线性近似，估计这一小步大约变了多少。

integral

积分

积分刻画连续小量的累计结果。

把无数个很小的量加起来，得到面积、路程或总量。

how they connect

它们之间的关系

微积分的核心不是一堆孤立公式，而是“从关系到局部变化，再从局部变化回到整体”的闭环。

函数描述整体关系
→
极限让近似变精确
→
导数得到瞬时变化率
→
微分估计一小步变化
→
积分累加小量成总量
导数来自极限 平均变化率\(\Delta y/\Delta x\)在\(\Delta x\to0\)时的极限，就是瞬时变化率。
微分来自导数 已知当前的率\(f'(x)\)，走一小步\(dx\)，变化量约为\(dy=f'(x)\,dx\)。
积分反过来累加 如果知道每一小步的量\(f(x)\,dx\)，把它们加起来就是\(\int f(x)\,dx\)。
interactive example

用 \(f(x)=x^2\) 做一个小实验

拖动 x，观察函数值、导数、微分和积分如何同时变化。这个例子足够简单，但能把五个概念全部串起来。

曲线与切线\(f(x)=x^2\)
读数\(dx=0.1\)
2.00
函数值：现在在哪里 \(f(x)=4.00\)
导数：变化有多快 \(f'(x)=4.00\)
微分：这一小步约变多少 \(dy\approx0.40\)
积分：0 到 x 的累计面积 \(\int_0^x t^2\,dt=2.67\)

这里固定 \(dx=0.1\)。导数 \(2x\) 是率；微分 \(dy\approx2x\cdot0.1\) 是这一小步的变化量；积分 \(\int_0^x t^2\,dt=x^3/3\) 是从 0 到 x 的累计面积。

一张表区分它们

遇到题目时，先判断它在问“关系、趋势、率、小量、总量”中的哪一个。

概念 核心刻画 典型问题 常见形式 直觉图像
函数 对应关系 输入 x，输出是多少？ \(y=f(x)\) 一台规则机器或一条曲线

applications

它们有什么用

只要有“变化”和“累计”，微积分就会出现。理工科里它通常是建模、优化和估计的底层语言。

v
物理运动

位置对时间求导得到速度，速度对时间积分得到路程。自由落体、汽车制动、轨道计算都靠它。

∂
工程误差

测量半径有小误差 \(dr\) 时，面积误差约为 \(dA=2\pi r\,dr\)，这就是微分估计。

min
优化问题

利润最大、成本最小、机器学习训练模型，本质上都在找导数为 0 或梯度下降的方向。

Σ
总量计算

水流量随时间变化，总用水量就是对流量积分；功率随时间变化，总耗电量也是积分。

记忆线索：函数看关系，极限看逼近，导数看快慢，微分看小步，积分看总和。
